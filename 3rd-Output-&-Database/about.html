Overview

In summary, the system collects vacancy status data from the parking spaces and display its output on this website with Google Sheets acts as a simplified database.

System Design

The smart parking system consist of four main process:

  • Arduino and ultrasonic sensors detects the presence/absence of car. The input data is sent to ESP8266 Wi-Fi module.
  • The ESP8266 Wi-Fi module connects to the assigned Wi-Fi connection, get the input data from Arduino, and sent it to Google Sheets.
  • The Google Sheets process the data from ESP8266 Wi-Fi module and store it as programmed using Google Apps Script.
  • Web application display the data stored on the Google Sheets as a website.
